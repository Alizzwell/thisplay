# thisplay


### bower 설치
```bash
$ npm install -g bower
```

### bower proxy 설정
.bowerrc 파일을 아래 내용을 작성
```bash
{
  "proxy": "http://yourProxy:yourPort",
  "https-proxy":"http://yourProxy:yourPort",
  "strict-ssl": false
}
```

### 의존성 설치
```bash
$ npm install
$ bower install
```

### grunt 설치
```bash
$ npm install -g grunt
```

### build
